Breakup Quest

project site | github repo

Breakup Quest is a graphical text adventure game. My team and I built this in 48 hours during the Global Game Jam in Edmonton, which is like a game development Hackathon. We built this using the Unity3D engine and us three programmers stayed organized using GitHub.
Our Global Game Jam page, where you can download and play the game, is linked above along with the GitHub repository.

Conecpt

Simply put, you are stuck in an arranged marriage, and you are trying to escape! Our team was looking to work on something funny and interesting and our artists came up with an idea for a reverse dating simulator, where the goal is to end a relationship, not build one.
